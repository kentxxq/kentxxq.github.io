`nacos` 是一个配置, 注册中间件.

要点:

- 免费, 开源
- 阿里巴巴支持, 且有商业版

### 运行

参考链接 [Nacos Docker 快速开始](https://nacos.io/zh-cn/docs/v2/quickstart/quick-start-docker.html)

```shell
git clone https://github.com/nacos-group/nacos-docker.git
cd nacos-docker
docker compose -f example/standalone-derby.yaml up
```
