{
    "Vue模板": {
        "scope": "vue",
        "prefix": "vvv",
        "body": [
            "<template>",
            "\t<div>",
            "\t\tdata",
            "\t</div>",
            "</template>\n",
            "<script setup lang='ts'>",
            "defineOptions({",
            "\tname: '$1'",
            "})",
            "</script>\n",
            "<style scoped>\n",
            "</style>"
        ],
        "description": "Vue模板"
    }
}
  • 当你输入 vvv, 就会自动出现模板
  • 默认光标会出现在 $1 的位置, 你可以继续添加 $2
  • 填写完 $1 后使用 tab 键, 会跳转到 $2
