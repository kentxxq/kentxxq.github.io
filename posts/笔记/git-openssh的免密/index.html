简介

我的使用 git 操作代码. 而 git 的通信会用到 openssh.

openssh 为了保证安全. 提供了私钥和公钥. 其中私钥可以密码加密, 保证安全性. 所以我就加密了.导致了什么问题呢?

每次我用到 git 的时候, 都提示我输入密码. 所以我今天就来配置 ssh-agent.

我暂时只用 windows, 所以下面主要是 windows 的配置.
