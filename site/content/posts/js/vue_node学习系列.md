> 以后可能会用到的知识

npm使用中用到的命令
===
```bash
# 我要使用element-ui
# npm i和npm install 是一样的意思
# -S, --save 安装包信息将加入到dependencies（生产阶段的依赖） 
# -D, --save-dev 安装包信息将加入到devDependencies（开发阶段的依赖），所以开发阶段一般使用它
npm i element-ui -S
# 这个babel插件可以按需导入饿了么的组件
# npm install babel-plugin-component -D
```
